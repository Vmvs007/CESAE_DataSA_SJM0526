Este repositório reúne as **resoluções das fichas práticas** desenvolvidas na UFCD de **Modelos Analíticos em Python**, lecionada no âmbito do curso de **Data Analyst** do **CESAE Digital**.

O objetivo é **facilitar o estudo**, **consolidar conhecimentos práticos** e **aplicar técnicas de análise, modelação e interpretação de dados utilizando Python**.

## 🔍 Temas Abordados

- Introdução ao Python aplicado à análise de dados
- Manipulação e exploração de dados
- Utilização de bibliotecas como NumPy e Pandas
- Visualização de dados com Matplotlib e/ou Seaborn
- Limpeza, tratamento e preparação de dados
- Análise exploratória de dados
- Introdução a modelos analíticos
- Regressão linear e interpretação de resultados
- Introdução à aprendizagem automática
- Avaliação simples de modelos
- Aplicação prática de técnicas analíticas em datasets

---
